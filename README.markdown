### This is an open-source app for referencing multiple files with full content to generate structured prompt for inference with large language models.
- Suitable for inferencing with models offering large context length
- Each file reference carries both file path and full file content. The file paths provides a full picture of the file/folder hierarchy to the model, especially for complext coding and writing projects.
